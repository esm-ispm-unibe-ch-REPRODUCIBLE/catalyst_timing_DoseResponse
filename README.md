How to reproduce the results presented in the article "Evaluating the optimal timing of anticoagulation after ischaemic stroke and atrial fibrillation: a Bayesian individual-patient data time-risk meta-analysis"
To reproduce the numerical results and the graphs, go through the following steps.

•	You need to have R and R-Studio installed.
•	Double click on CATALYST_timing.Rproj file
•	From within R-Studio, open the file 00_master.R and run it.

You need to have the data from catalyst or the synthesized data so that the codes can be run
